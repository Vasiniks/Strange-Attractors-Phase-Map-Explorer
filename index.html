What is the Lyapunov Map?
Each pixel represents an initial condition. Its color shows the Lyapunov exponent λ — a measure of how quickly nearby trajectories diverge.

λ > 0 (orange/warm): chaotic — exponential divergence.
λ < 0 (blue/cool): stable — trajectories converge.

The map axes can be any two system parameters or initial conditions. Choose them in the Axes controls below.
Equation Editor [E]
Enter any 3D autonomous ODE. Use x y z as state variables. Any other letter or Greek symbol becomes a slider parameter automatically.

Choose from built-in attractors: Lorenz, Rössler, Thomas, Halvorsen, Dadras, Aizawa, Chen.

Renders the current map at a custom resolution and copies all parameters in LaTeX or plain text for papers and presentations.
